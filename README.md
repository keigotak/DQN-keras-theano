# DQN-keras-theano
This project implements [DQN](https://deepmind.com/research/dqn/) with:
 * python 2.7.11
 * keras  1.0.4
 * theano 0.9.0 dev1
 * pygame 1.9.2 a0
 
If you start learning with this project run "Clean.py" or "Gleaning.py".
